## §0 컨텍스트 · 페르소나 · ID 규약

- 컨텍스트: ContextA / ContextB / ... (도메인 영역 분할)
- 페르소나: 정책서 §페르소나 그대로 사용. 한글 → 영문 mapping이 필요한 경우 본 섹션에서 명시
  (예: 시청자(회원) → `시청자회원`, 시청자(비회원) → `시청자비회원`)
- 화면 ID: `<컨텍스트>_<페르소나>_NN` (페르소나별 NN은 01부터 2자리 패딩)
- 액션 ID: `<페르소나>:Action:N` (페르소나별 1부터 증가)
- 색상 범례:
  - 빨강(error): 에러/실패/충돌
  - 노랑(unclear): 정책서 미정의 — 회의 결정 필요
  - 보라(auth): 인증/권한 게이트
  - 점선(system): 시스템 트리거
